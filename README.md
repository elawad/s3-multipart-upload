# S3 Multipart Upload #

> Testing AWS Multipart file upload with Hapi.

### Setup ###
* Create a .env file with the following:
````javascript
AWS_ACCESS_KEY_ID     = ''  // AWS access key
AWS_SECRET_ACCESS_KEY = ''  // AWS secret
S3_BUCKET             = ''  // bucket name
S3_FOLDER             = ''  // (optional) folder name
````
* npm install
* npm start

### Tech Stack ###
* Node / Hapi
* AWS SDK
